# /iterate 명령어

GAPEVI 반복(Iterate) 단계를 실행합니다.

---

## 사용법

```
/iterate [복귀 단계]
```

---

## 목적

검증 결과에 따라 이전 단계로 복귀하여 개선합니다.

---

## 실행 내용

### 1. 문제 분석

| 문제 유형 | 복귀 단계 |
|----------|----------|
| 타입 오류 | → `/execute` |
| 빌드 실패 | → `/execute` |
| 테스트 실패 | → `/execute` |
| 요구사항 변경 | → `/analyze` |
| 새 정보 발견 | → `/gather` |
| 설계 문제 | → `/prioritize` |

### 2. 복귀 실행

```
문제 발견
    │
    ▼
┌───────────────────────────────────┐
│  복귀 단계 결정                    │
├───────────────────────────────────┤
│                                    │
│  코드 오류 ────→ /execute          │
│  설계 문제 ────→ /prioritize       │
│  분석 부족 ────→ /analyze          │
│  정보 부족 ────→ /gather           │
│                                    │
└───────────────────────────────────┘
    │
    ▼
해당 단계부터 재실행
```

### 3. 재검증

복귀 후 다시 `/verify` 실행

---

## 출력 형식

```markdown
## 반복 결정

### 발견된 문제
- [문제 설명]

### 원인 분석
- [원인]

### 복귀 단계
**→ `/[단계명]`**

### 수정 계획
1. [수정 사항 1]
2. [수정 사항 2]
```

---

## 사용 예시

```bash
# 타입 오류로 인한 복귀
/iterate execute
# → /execute 단계로 복귀하여 코드 수정

# 요구사항 변경으로 인한 복귀
/iterate analyze
# → /analyze 단계로 복귀하여 재분석

# 정보 부족으로 인한 복귀
/iterate gather
# → /gather 단계로 복귀하여 추가 정보 수집
```

```
│                                          │

```
