# /gather 명령어

GAPEVI 수집(Gather) 단계를 실행합니다.

---

## 사용법

```
/gather [대상 설명]
```

---

## 목적

작업에 필요한 모든 정보를 수집합니다.

---

## 실행 내용

### 1. 요구사항 수집

| 활동 | 설명 |
|------|------|
| 요청 분석 | 사용자 요청의 핵심 파악 |
| 명확화 질문 | 모호한 부분 확인 (필요 시) |

### 2. 컨텍스트 수집

| 활동 | 에이전트 | 설명 |
|------|----------|------|
| 관련 코드 탐색 | **Explore** | 기존 구현 분석 |
| 파일 구조 파악 | **Explore** | 프로젝트 구조 확인 |
| 패턴 분석 | **Explore** | 기존 패턴/컨벤션 파악 |

### 3. 제약사항 파악

| 활동 | 설명 |
|------|------|
| 기술적 제한 | Phaser, TypeScript 제약 |
| 의존성 확인 | 필요 라이브러리/모듈 |

---

## 출력 형식

```markdown
## 수집 결과

### 요구사항
- [요청 핵심 내용]

### 관련 파일
| 파일 | 역할 |
|------|------|
| [파일 경로] | [역할 설명] |

### 기존 패턴
- [발견된 패턴]

### 제약사항
- [확인된 제약]
```
