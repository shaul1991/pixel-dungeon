# /execute 명령어

GAPEVI 실행(Execute) 단계를 실행합니다.

---

## 사용법

```
/execute [작업 번호 또는 설명]
```

---

## 목적

계획된 작업을 실제로 구현합니다.

---

## 실행 내용

### 1. 코드 구현

| 에이전트 | 담당 영역 | 산출물 |
|----------|----------|--------|
| **Entity** | `src/entities/` | 엔티티 클래스 |
| **System** | `src/systems/` | 게임 로직 |
| **Scene** | `src/scenes/` | 씬 통합 |
| **UI** | `src/ui/` | UI 컴포넌트 |
| **Asset** | `public/assets/` | 스프라이트, 타일셋 |

### 2. 테스트 코드 작성 (필수)

| 상황 | 테스트 요구 |
|------|------------|
| 새 기능 | 로직 단위 테스트 |
| 버그 수정 | 버그 재현 테스트 → 수정 → 통과 확인 |

**테스트 대상**:
```
✅ 필수: systems/ 순수 로직
✅ 권장: entities/ 상태 로직 (모킹)
❌ 제외: Phaser 의존 코드 (수동 테스트)
```

### 3. 구현 순서

```
1. 타입 정의 (types/)
2. 엔티티 구현 (entities/)
3. 시스템 구현 (systems/)
4. 씬 통합 (scenes/)
5. UI 연결 (ui/)
6. 테스트 코드 (*.test.ts) ← 필수!
```
