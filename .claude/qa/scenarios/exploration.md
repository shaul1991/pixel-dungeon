**버전**: v0.1.0
**소요 시간**: 10분
**목적**: 맵 탐험 및 이동 관련 기능 검증
**업데이트**: 2026-02-03

---

## 개요

게임의 탐험 요소를 검증합니다:
- 플레이어 이동
- 타일맵 충돌
- NPC 상호작용
- 맵 오브젝트

---

## 사전 준비

```bash
make dev
# http://localhost:5173
# New Game 시작
```
