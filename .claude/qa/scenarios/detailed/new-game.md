# 시나리오: 새 게임 시작

**ID**: SC-001
**소요 시간**: 3분
**우선순위**: 높음
**담당 영역**: 씬 전환, 초기화

---

## 목적

게임을 처음 시작할 때 모든 초기화가 정상적으로 이루어지는지 검증합니다.

---

## 사전 조건

1. [x] 개발 서버 실행 (`make dev`)
2. [x] 브라우저 캐시 삭제 (권장)

---

## 테스트 단계

### Step 1: 페이지 로딩

**액션**:
1. 브라우저에서 `http://localhost:5173` 접속
2. 개발자 도구 (F12) 콘솔 탭 열기

**예상 결과**:
- 콘솔에 에러 없음
- 로딩이 완료됨

---

### Step 2: Boot → Preload 전환

**액션**:
1. 페이지 로딩 직후 화면 관찰

**예상 결과**:
- BootScene이 잠시 표시됨 (검은 화면 또는 로고)
- PreloadScene으로 자동 전환
- 에셋 로딩 진행

---

### Step 3: Preload → Menu 전환

**액션**:
1. 로딩 완료 대기

**예상 결과**:
- 로딩 완료 후 MenuScene으로 자동 전환
- 메인 메뉴 화면 표시
- "Pixel Dungeon" 타이틀 표시
- "New Game" 버튼 표시

---

### Step 4: 게임 시작

**액션**:
1. "New Game" 버튼 클릭

**예상 결과**:
- GameScene으로 전환
- 타일맵 표시 (마을 맵)
- 플레이어 캐릭터 표시
- NPC 및 몬스터 배치됨

---

### Step 5: 초기 상태 확인

**액션**:
1. 화면 전체 관찰
2. 콘솔 확인

**예상 결과**:
- 플레이어 초기 위치 정상 (맵 중앙 부근)
- 플레이어 idle 애니메이션 재생
- 콘솔에 에러 없음
- DEBUG=true 시 디버그 정보 표시
