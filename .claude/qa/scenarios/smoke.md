# 스모크 테스트 (Smoke Test)

**버전**: v0.1.0
**소요 시간**: 5분
**목적**: 핵심 기능의 기본 동작 확인
**업데이트**: 2026-02-03

---

## 개요

스모크 테스트는 게임의 핵심 기능이 기본적으로 동작하는지 빠르게 확인합니다.
코드 변경 후 커밋 전, 또는 빌드 후 기본 검증에 사용합니다.

---

## 사전 준비

```bash
# 자동화 테스트 먼저 실행 (권장)
make qa-auto

# 개발 서버 시작
make dev

# 브라우저에서 열기
# http://localhost:5173
```

- **전체 통과 (13/13)**: 다음 작업 진행 가능
- **11-12/13**: 경미한 이슈, 문서화 후 진행 가능

## 실패 시 조치

1. 콘솔 에러 확인 (F12 → Console)
2. 실패 항목 재현 시도
3. 관련 코드 확인:
   - 게임 시작: `src/scenes/MenuScene.ts`
   - 플레이어 이동: `src/entities/Player.ts`, `src/systems/InputController.ts`
   - NPC: `src/entities/NPC.ts`, `src/ui/DialogBox.ts`
   - 전투: `src/scenes/BattleScene.ts`, `src/systems/BattleSystem.ts`
